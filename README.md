# DroughtMap Edu
**Peta Edukasi Pemantauan Kekeringan Berbasis Data Geospasial**

Website ini dalah dashboard interaktif untuk memantau kekeringan secara real-time menggunakan model NDDI (Normalized Difference Drought Index) dengan data satelit Landsat dan MODIS melalui Google Earth Engine.

## Fitur Utama
- **Real-time Monitoring:** Integrasi langsung dengan Google Earth Engine.
- **Tampilan Profesional:** Desain "Geospatial" dengan tema gelap dan animasi interaktif.
- **Responsif:** Tampilan optimal di Desktop (Single View) dan Mobile (Scrollable).

## Cara Menggunakan
1. Buka website.
2. Pilih Tahun dan Bulan pada panel kiri peta (dalam iframe).
3. Analisis peta sebaran kekeringan yang muncul.
